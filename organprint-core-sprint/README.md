# OrganPrint — Riwayat'25 Core Sprint (Internal PS)

**Competition:** Riwayat'25 · **Core Sprint** · E-Cell · IIT Kanpur  
**Submission date:** November 3, 2025  
**Type:** Internal inter-hall problem statement (not a course project)

## Problem

India's organ transplant crisis: **500,000+ annual deaths** from organ failure, **1:50** supply–demand ratio, **₹1.5–2 Cr** per transplant, **40–50%** five-year rejection rates, **5–10 year** wait lists.

## Proposed solution — OrganPrint

Patient-specific **3D bioprinted organs** using:

- **4-nozzle extrusion bioprinting** — structural scaffold, vascular endothelial cells, organ parenchyma, ECM/growth factors (200–150 μm nozzles)
- **Vascularized hydrogel printing** — macro-channels (200–500 μm) + micro-channels (50–100 μm); validated against Nature 2025 / Chemical Society Reviews 2024 literature
- **AI organ design** — U-Net 3D segmentation on patient MRI/CT → print path in <1 hour; >95% geometry accuracy
- **Autologous stem-cell integration** — target **<5% rejection**, **85–95%** post-print cell viability

## MVP targets

- First functional printed kidney within **18 months**
- Print time **2–4 hours** per organ at **₹30–50L** (vs **₹1.5–2 Cr** conventional transplant)
- Scale target: **100,000+ transplants/year by 2030**
